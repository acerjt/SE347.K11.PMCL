Two Column Flexible

Ver 2.3

This is a very easy page layout which you have probaly seen on sites such as ALA (A List Apart). Each column is created with simply two div tags.

The left column uses float left and has percentage widths allowing both columns to flow as the browser window changes shape.

This template has been tested in:

  • Mozilla 1.5b & FireFox 0.8
  • Internet Explorer 6
  • Nestscape 7.1
  • Opera 7.11
