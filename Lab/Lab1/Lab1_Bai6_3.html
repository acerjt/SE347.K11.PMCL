Two Column Flexible With Header

Ver 2.3

This is the same as the two Column Flexible but with the addition of a header div for top navigation or banner.

The left column uses float left and has percentage widths allowing both columns to flow as the browser window changes shape.

This template has been tested in:

  • Mozilla 1.5b & FireFox 0.8
  • Internet Explorer 6
  • Nestscape 7.1
  • Opera 7.11

change in this version:

  1. Define zero padding for the body to fix a problem in Opera

Right column

The right column can be used for navigation links, search box etc.
